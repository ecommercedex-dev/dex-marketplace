## Seller Perspective

### "Why should I use the platform instead of just WhatsApp?"

**Answer (without saying it):**
1. **Notifications** - I know immediately when someone orders
2. **Organization** - All orders in one place, not scattered WhatsApp chats
3. **Professional** - Looks more legit than random WhatsApp messages
4. **Auto stock** - Don't have to manually update inventory
5. **Order history** - Can track all past sales

### "What if I want to reject an order?"

**Answer:**
- Click "Reject" button
- Order moves to trash
- Can restore later if needed
- Buyer gets notification

### "What if buyer doesn't respond on WhatsApp?"

**Answer:**
- You already accepted, so you have their contact
- Can call/text/WhatsApp multiple times
- If no response after 24hrs, can reject order
- Platform tracks everything
